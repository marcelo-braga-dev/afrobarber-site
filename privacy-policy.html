Política de Privacidade

Última atualização: 2026

Esta Política de Privacidade descreve como o jogo AfroBarber coleta, usa e protege as informações dos usuários. Ao utilizar o aplicativo, você concorda com os termos descritos abaixo.

1. Informações coletadas

O AfroBarber coleta apenas as informações necessárias para funcionamento do jogo e melhoria da experiência do usuário:

  • Nome de usuário escolhido pelo jogador
  • Pontuação e dados de ranking
  • Dados de progresso (níveis, missões, conquistas)
  • Informações básicas do dispositivo (modelo, sistema operacional)
  • Dados de uso do aplicativo (tempo de uso, interações)

2. Uso de serviços de terceiros (Unity Services)

O jogo utiliza serviços da Unity (Unity Services), que podem coletar e armazenar dados para funcionalidades como:

  • Salvamento em nuvem (Cloud Save)
  • Ranking de jogadores (Leaderboards)
  • Autenticação básica do jogador
  • Analytics para melhoria do jogo

Esses serviços podem coletar dados técnicos automaticamente. Para mais informações, consulte a política da Unity:

https://unity.com/legal/privacy-policy

3. Informações NÃO coletadas

O AfroBarber NÃO coleta:

  • Dados pessoais sensíveis
  • Localização precisa do usuário
  • Mensagens ou chat (o jogo não possui comunicação online entre usuários)
  • Dados biométricos

4. Uso das informações

As informações coletadas são utilizadas exclusivamente para:

  • Salvar o progresso do jogador
  • Exibir rankings e pontuações
  • Melhorar a experiência do usuário
  • Corrigir erros e falhas
  • Otimizar desempenho do jogo

5. Compartilhamento de dados

Os dados podem ser compartilhados apenas quando necessário para:

  • Funcionamento dos serviços da Unity
  • Cumprimento de obrigações legais

Não vendemos nem compartilhamos dados pessoais com terceiros para fins comerciais.

6. Armazenamento e segurança

Os dados podem ser armazenados localmente no dispositivo do usuário e/ou em servidores seguros da Unity. São adotadas medidas para proteger as informações contra acesso não autorizado, alteração ou destruição.

7. Público e classificação etária

O AfroBarber é adequado para todas as idades e não contém:

  • Violência
  • Sangue
  • Drogas
  • Conteúdo impróprio

8. Direitos do usuário

O usuário pode solicitar:

  • Exclusão de dados
  • Esclarecimentos sobre uso de dados

Para isso, entre em contato pelo e-mail informado abaixo.

9. Retenção de dados

Os dados serão mantidos apenas pelo tempo necessário para cumprir as finalidades descritas nesta política, ou até solicitação de exclusão pelo usuário.

10. Alterações nesta política

Esta política pode ser atualizada periodicamente. Recomendamos que o usuário revise esta página regularmente.
